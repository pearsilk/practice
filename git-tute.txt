Simple refreshers about GitHub

- git init to initialize a folder/repo
- git add to add a file to staging
- git commit -m "message here" to commit file
- git status

- to add a local repo to github:
	- go online and create new repo on site
	- git push origin [branch]
	**N.B.: have to 'git add' and 'git commit -m' a file(s) first
